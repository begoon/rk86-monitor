# Оригинальный Монитор Радио-86РК

Дизассемблированный и аннотированный листинг оригинального 2КБ ПЗУ Монитора
для Радио-86РК — советского любительского компьютера на базе процессора
Intel 8080 (КР580ВМ80А).

Монитор обеспечивает базовый ввод-вывод, работу с памятью, магнитофоном
и простой командный интерфейс.

Используется в эмуляторе [rk86.ru](https://rk86.ru).

Смысл проекта — не менять код, а максимально покрыть его комментариями.

## Файлы

- `monitor.asm` — основной аннотированный листинг (комментарии на русском), компилируется в бинарник, идентичный оригинальному ПЗУ
- `original/mon32.bin` — оригинальный бинарный образ ПЗУ (2048 байт, эталон)
- `alt/MonitorRK16.txt` — альтернативный дизассемблированный листинг

## Сборка

```
make          # компиляция + сравнение с оригинальным ПЗУ
make build    # только компиляция
make test     # сравнение результата с original/mon32.bin
make clean
```

Ассемблер: [`asm8080`](https://www.npmjs.com/package/asm8080) из npm (`npx --yes asm8080` или `bunx --bun asm8080`). Требуется Node.js или [Bun](https://bun.sh).

## Соглашения

- Комментарии на русском языке
- Константы аппаратных регистров (equ) в UPPER_CASE (например, `PORT_A_KBD`, `CONTROL_VG75`)
- Имена переменных в памяти в lower_case (например, `cursor_addr`, `ruslat_flag`)
- Метки кода в lower_case (например, `entry_start`, `run_D_command`)
- Каждая метка кода имеет комментарий с адресом: `label_name:  ; F800`
- Скомпилированный бинарник должен быть побайтово идентичен `original/mon32.bin`

## CI

GitHub Actions (`.github/workflows/test.yaml`) запускает `make` при каждом push — компилирует и сравнивает с оригинальным бинарником.
